# Music_11

Website nghe nhạc trực tuyến được xây dựng nhằm mô phỏng các nền tảng streaming phổ biến như Spotify, Zing MP3 và SoundCloud.

## Giới thiệu

Music_11 là đồ án phát triển hệ thống nghe nhạc trực tuyến với mục tiêu xây dựng một nền tảng có khả năng:

- Nghe nhạc trực tuyến
- Quản lý thư viện bài hát
- Phát nhạc realtime
- Quản lý người dùng
- Tìm kiếm và khám phá nhạc
- Giao diện hiện đại, dễ sử dụng

Dự án được xây dựng nhằm áp dụng kiến thức về phát triển web fullstack, thiết kế cơ sở dữ liệu và xây dựng API.

## Cài đặt và chạy dự án

### Clone repository

```bash
git clone https://github.com/DIIX-Program/music_11.git
cd music_11
```

### Cài đặt dependencies

```bash
npm install
```

### Chạy frontend

```bash
npm run dev
```

### Chạy backend

```bash
npm start
```
